# Analysis Scripts

Python scripts for reproducing bootstrap validation and PERMANOVA analysis.

## Requirements

- Python 3.9+
- Dependencies in `requirements.txt`

```bash
pip install -r requirements.txt
```

## Scripts

| Script | Purpose | Output |
|--------|---------|--------|
| `bootstrap_obs2.py` | Observation 2: Style > Provider variance | `bootstrap_obs2_results.json` |
| `bootstrap_obs5.py` | Observation 5: Shift decline with loop | `bootstrap_obs5_results.json` |
| `bootstrap_obs8.py` | Observation 8: Shape-scale correlations | `bootstrap_obs8_results.json` |
| `permanova_analysis.py` | PERMANOVA variance decomposition | `permanova_results_*.csv` |

## Usage

Each script can be run standalone:

```bash
# Run bootstrap validation for Observation 2
python bootstrap_obs2.py --data ../data/variance/variance_ratio_by_loop.csv

# Run PERMANOVA analysis
python permanova_analysis.py --data ../data/manifold/umap_coordinates.csv
```
